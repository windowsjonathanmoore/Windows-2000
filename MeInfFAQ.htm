FAQ for Installing Drivers on Windows Millennium

  1. Where can I get more information on writing infs for Millennium?

    Check the INF Guide topic under Millennium DDK\Installing Drivers on Windows Millennium\INF Guide for Windows Millennium in the online documentation.

  2. I want to disable version checking and always copy my file. How can I do this?

    This is set by a flag in a CopyFile section. The values of these flags are shown in the online help under Millennium DDK\Inf Guide for Windows Millennium\Default Install\Copy Files Section (CopyFiles).

  3. I want to copy a file only if it does not exist on a user's system. How can I do this?

    Please see question 2 above.

  4. How do I avoid overwriting critical system files?

    Microsoft has a new feature, System File Protection (SFP) specifically designed to prevent overwriting shared system files. Check the online documentation under Millennium DDK\Installing Drivers on Windows Millennium\System File Protection.

  5. What files are protected under SFP?

    A list is provided for you in the online documentation. See the "protected file list" link in Millennium DDK\Installing Drivers on Windows Millennium\System File Protection.

  6. How do I disable System File Protection.

    System File Protection (SFP) cannot be disabled. Check the online documentation under Millennium DDK\Installing Drivers on Windows Millennium\System File Protection for more info.

  7. How do I write an inf to install a WDM Driver on both Windows 2000 and Windows Me?

    Please see the example in the section "Writing a Cross-Platform INF to Install USB Printer Drivers" located in the online documentation under Millennium DDK\Universal Serial Bus\USB Printing.

  8. Where can I get a list of install directories or Logical Disk Identifiers (LDIDs)?

    Check the DestinatonDirs section in the online documentation under Millennium DDK\Intalling Drivers on Windows Millennium\INF Guide for Windows Millennium.

  9. How can I get my drivers signed?

    Check the Driver Signing Summary section in the online documentation under Millennium DDk\Installing Drivers on Windows Millennium\Driver Signing.

  10. How do I get a digital signature for my driver?

    Check the online documentation for the section, How to Get Digital Signatures for Windows Drivers, under Millennium DDK\Installing Drivers on Windows Millennium\Driver Signing.
