Metabase Backup Delete Utility

Obviously, you want to be very careful about deleting backup location sets. However, sometimes, after months of diligent metabase backups, you will find that your hard drive is overflowing with old metabases. This tool illustrates how to delete metabase backups if you really decide that it's time to discard some of those old versions.

DeleteBackup, like all the other metabase backup methods, is accessible through the IIS://LocalHost node. The method takes as parameters the name of the location set and the version number. Also like the other backup methods, you can specify constants for the version number. The most important, &HFFFFFFFE, indicates that IIS should delete the highest numbered backup. It is advised that you use this constant with caution.

This sample is provided for educational purposes only. It has not been tested in a production environment, and Microsoft® will not provide technical support for it.
