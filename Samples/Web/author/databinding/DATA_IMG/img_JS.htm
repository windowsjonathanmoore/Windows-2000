The IMG Tag

The SRC element of the IMG tag can be bound to the current record of a data set. The bound data should consist of a URL to an image file.

Here is an example of an IMG tag that uses current record binding. Clicking on the arrow buttons will move the current record. This results in the bound URL changing and the bound image displaying the image from the new URL.
