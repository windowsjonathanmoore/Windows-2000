The LABEL Tag

 Close This Sample

The content of the LABEL tag can be data bound. Clicking on the content results in the associated controls obtaining the focus. Below is an example of labels bound to the current record of a data source. Clicking on the arrow buttons will move the current record. This results in the label displaying the data from the current record.

The data the label binds may contain HTML. Using the DATAFORMATAS attribute, the HTML can be displayed for the label. Clicking anywhere on the HTML will result in the associated control obtaining the focus.

Data bound labels cannot be used within repeated tables.
