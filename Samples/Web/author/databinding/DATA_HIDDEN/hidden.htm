The HIDDEN Tag

 Close This Sample

Although they are not visible, the input hidden element can be data bound. Binding is to the VALUE attribute of the hidden element.

Click on the button below to display the value of the hidden element. It is bound to the Symbol column in the data set.

Clicking on the arrow buttons will move the current record. This results in the hidden changing to display the values from the current record. After pressing the arrow buttons, click on the Show Value button again. Notice that the value now matches the value from the current record.

Hiddens can also be used in repeated tables. Click on the buttons in the table to get the value of the HIDDEN tag.
