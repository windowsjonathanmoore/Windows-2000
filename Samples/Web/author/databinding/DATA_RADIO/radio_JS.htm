The RADIO Button

A radio button or group of radio buttons can be bound to a column in the data set. The binding stores the VALUE attribute from the selected radio button. If the value in the data set matches no VALUE attributes on the radio button group, then no radio buttons are selected.

Below is an example of a radio button group with a current record binding. Clicking on the arrow buttons will move the current record. This results in the radio button with the matching value being selected.
