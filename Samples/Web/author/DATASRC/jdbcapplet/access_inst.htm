Using the JDBC Data Source Applet with Microsoft Access

Description

If you're viewing this page, you're presumed to have installed the JDBC software from the JDBC Applet introduction page. If not, click here to go install the software.

This page provides instructions to use the software to access data stored in a Microsoft Access database on your local system.

Instructions for Microsoft Access users

These instructions assume you have Microsoft Access installed on your system along with the desktop ODBC drivers. It also assumes that you installed the sample database, Northwind.mdb.

  1. Create an ODBC DSN to the Northwind database. Name this DSN Northwind.
  2. Press the refresh button on the browser.
  3. When you scroll down, you should now be viewing a list of products from the Northwind Trader's database including the product name, the name of the company supplying the product, and the product category.
