Using an ODBC Compliant Database

Description

If you're viewing this page, you are presumed to have installed the JDBC software from the JDBC applet introduction page. If not, click here to go install the software.

This page provides instructions to use the software with databases other than Microsoft Access or Oracle.

If you have an ODBC-compliant database, you can modify the sample code included on the Access Instructions page. The following instructions provide the general tasks required to use the applet with your database.

  1. create an ODBC DSN to your database,
  2. store the HTML for one of the Access example page,
  3. edit the page and find the APPLET tag for the JDBC Applet. It has the attribute ID=DSA1.
  4. Change the VALUE attribute of the PARAM with NAME=dbURL to be: jdbc:jet:DSN where DSN is the name of the DSN created in step i) above,
  5. Edit the query to retrieve data from the database pointed to by the DSN.
  6. Modify any additional Id, password, or allow* PARAMs as appropriate.
  7. Change the references to DATAFLD within the TABLE below the applet to the column names retrieve from your query,
  8. Browse the page using IE4.0.

Using a JDBC Driver

If you have a JDBC driver to a database, you'll need to rebuild the applet using the source code provided. Once the applet is rebuilt, you can modify the sample code included on the Access page. Follow the instructions above.
