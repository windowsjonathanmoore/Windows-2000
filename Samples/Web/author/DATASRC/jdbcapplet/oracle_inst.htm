Using the JDBC Data Source Applet with Oracle

Description If you're viewing this page, you're presumed to have installed the JDBC software from the JDBC Applet introduction page. If not, click here to go install the software.

This page provides instructions to use the software to access data stored in the DEMO database installed with Oracle.

Instructions for all Oracle users

These instructions assume you have access to an Oracle database, either on your machine or through your network. You must also have an Oracle ODBC driver installed on your machine that is configured to access this database.
Note: In most cases this requires Oracle SQL*Net.
  1. Save this page to a file on your local system.
  2. Edit the page and change the following two lines inside the applet tag:
    <param name="user" value="nothing">
    <param name="password" value="nothing">
    Change the values "nothing" to your UserID and password to access the Oracle database.
  3. Save the file.
  4. Proceed to the specific instructions below.

Instructions for Oracle users with the DEMO database

  1. Create an ODBC DSN to the DEMO database. Name this DSN OracleDemo.
  2. Run the page you saved in step 3.
  3. You should now be viewing a list of customers from the DEMO database table CUSTOMER.

Instructions for Oracle users without the DEMO database

If you do not have the DEMO database installed:
  1. change the sqlStatement attribute on the JDBC Applet to be a query to a database accessible from your system.
  2. Change the DATAFLD attributes in the table to match the field names returned from your query.
  3. Change the OracleDemo DSN on the dbURL PARAM of the applet to the DSN defined on your system.
  4. Save the page.
  5. Run the page.
  6. You should now be viewing the data from your query.
