The final release of Microsoft Internet Explorer 4 and 5 include Dynamic HTML, the next generation of HTML that moves beyond static text and graphic images to interactive Web pages and live business applications. In addition to making Web page design easier and more precise, Dynamic HTML enriches the Web experience by allowing users to interact with multimedia content. A key criteria of an interactive Web page is that page elements readjust automatically if a graphic is resized, as demonstrated in this example.

This sample lets users replace a large graphic with a smaller graphic. Text surrounding the graphic re-flows automatically as a result.
