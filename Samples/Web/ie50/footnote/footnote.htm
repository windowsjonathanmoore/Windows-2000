A footnote tag

Thanks to the use of the new DHTML component model Use Internet Explorer 5 to see the behavior in action , it is now possible to attach behavior to any tag. In particular it is possible to attach behavior to XML tags in the HTML document the XML namespace can be declared in the head of the document.

This can be used to create a FN:FOOTNOTE tag as shown in this document. This tag inserts a auto-numbered footnote reference and pushes the text it contains to the end of the document as a footnote (or rather, an endnote) does. It also automatically sets up a tooltip showing the text of the footnote when the mouse is over the reference. The implementation of this behavior is only a few lines of JavaScript Note that the current implementation does not support dynamic addition or removal of footnotes after the page has been downloaded, but it would not be very difficult to add.. The particular implementation shown here supports two attributes to customize the format of the references and of the text of the footnote.

The syntax for this tag is the following:
...
<FN:FOOTNOTE REFFORMAT=string TEXTFORMAT=string>
text of the footnote
</FN:FOOTNOTE>
...

REFFORMAT: specifies the format of the reference to the footnote. Default is "(#)".
TEXTFORMAT: specifies the format of the footnote header placed just before the text of the footnote. Default is "#/".
The string arguments are any string of character where '#' represents the footnote number, i.e., "(see #)"
A link to the footnote text is automatically wrapped around the footnote reference.
