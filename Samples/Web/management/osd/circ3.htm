Circle ActiveX Control Channel Demo

Below is a circle in an object tag. It requires the Circle ActiveX Control (initially just build 1). After subscribing to the Circle software update channel below and updating it (after build 2 is available on the server), returning to this page will automatically update you to build 2 of the Circle control. Please note that there is no visual difference between the two versions so you will have to look in the Downloaded Program Files directory to verify that the new version has been installed.

Add this channel to automatically receive updates of the Circle ActiveX control.
