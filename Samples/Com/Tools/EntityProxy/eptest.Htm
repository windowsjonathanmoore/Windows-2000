The EPTest Sample

The EPTest sample uses a number of the components and the Entity Proxy Data Services to provide a simple random order entry test for the SQL Server 7.0 sample "Northwind" database.

Build the DataServices sample according to the instructions.

  1. Open the EPTest.vbp project in Visual Basic 6.0.
  2. Build the application.

Remarks

The EPTest sample shows how to use the components in a realistic scenario.
