This document contains late-breaking information about the OLE DB Simple Provider (OSP) Toolkit samples, which have been created and tested for compatibility with this version of the Microsoft Data Access Components (MDAC).

For information about the latest version of MDAC, see http://www.microsoft.com/data/download.htm.

For detailed information about the samples, see the OLE DB documentation within the MDAC SDK, which is contained within the Platform SDK. When building samples, make sure MDAC SDK libraries are used and not similar libraries installed by other Microsoft products.

Technical Issues

Compilation of this sample requires a new version of the file Simpdata.h. Details are explained in the MDAC SDK Readme (MDACSDKReadme.htm).

The MDAC SDK Readme also contains information about OSP methods for Visual J++ and Internet Explorer 4.0.

Known Bugs and Limitations

There are no known bugs or limitations at this time.
