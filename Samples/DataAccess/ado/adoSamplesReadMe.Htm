This document contains late-breaking information about the ADO samples, which have been created and tested for compatibility with this version of the Microsoft® Data Access Components (MDAC).

For information about the latest version of MDAC, see http://www.microsoft.com/data/download.htm.

For detailed information and setup instructions for the samples, see the ADO documentation within the MDAC SDK, which is contained within the Platform SDK. When building samples, make sure MDAC SDK libraries are used and not similar libraries installed by other Microsoft products.
