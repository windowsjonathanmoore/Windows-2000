1. What ProgEdit Does & How

    The ProgEdit sample connects to an SMS Provider namespace
    and enumerates the instaces SMS_Package and their related 
    SMS_Program instances.  This sample allows the user to Create,
    Modify, and Delete instances of SMS_Program and attach them to
    an instance of SMS_Package.  ProgEdit demonstrates how to 
    work with instances, and display their properties in a 
    simple user interface.

2. Running the Sample

    Open up the ProgEdit.vbp file in Visual Basic and run the program.
    Press connect and fill in the information fields on the connection
    box.  After the Package\Program list box has populated, select a program
    to view it properties.  Use the New Program, Save Program, and Delete
    Program buttons to Create, Modify, and Delete instances of SMS_Program

3. Requirements

    Requires: Visual Basic 6.0 or greater to compile and run
    Runs on: NT 4.0+
    Windows Management Instrumentation (WMI) must be installed.

4. Compiling the Sample

    Note: Compiling the Sample App requires the WMI SDK to be installed.

    Open up the .VBP Project file and select Run -> Start or press F5
