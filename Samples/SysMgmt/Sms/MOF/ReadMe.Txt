1. What these mof files do & how.  

    These mof files are examples of how you can use a simple text file
     to create instances in the SMS Provider namespace.

    For more information on the syntax of mof files and how they work refer to the WMI SDK.


2. Running the Sample

    Each of the mof files can be compiled using Mofcomp.exe.  Open the mof files and read the
     inline comments for exact instructions.

3. Requirements

    Runs on:	NT 4.0
    Windows Management Instrumentation (WMI) must be installed.
