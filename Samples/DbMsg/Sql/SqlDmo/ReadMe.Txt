 SQL DISTRIBUTED MANAGEMENT OBJECTS (SQL-DMO) SAMPLE PROGRAMS

This file describes the SQL-DMO sample programs included with

SQL Server Books Online.

32-BIT VISUAL BASIC SAMPLES

SQL-DMO samples include the following samples written for 32-bit
Microsoft Visual Basic(R).


c:\mssql7\devtools\samples\sqldmo\vb\explore

Sample SQL-DMO object browser using 32-bit Visual Basic.
Includes the following files:
    explore.mak
    explore.frm
    explore.bas

c:\mssql7\devtools\samples\sqldmo\vb\idxtest

Sample index testing program using 32-bit Visual Basic. Includes
the following files:
    idxtest.vdp
    idxtest.frm
    getparam.frm
