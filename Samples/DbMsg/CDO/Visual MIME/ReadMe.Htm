Visual MIME

Visual MIME is a sample utility that demonstrates the capabilities of Microsoft Collaboration Data Objects (CDO) for Windows 2000.  

Using this utility you can view and manipulate the MIME structure of messages stored as .eml files on disk.  You can create .eml files by using Microsoft Outlook Express to save existing messages to disk or by using Visual MIME.  An example .eml message was also included (microsoftmhtml.eml) for you to view using Visual MIME. 

This sample utility was created using Microsoft Visual Basic 6.0.  Visual Basic must be installed in order to run this sample.
