This sample application allows you to view 
and make changes to the MSMQ-MQSeries Bridge 
using Wmi.  To run the sample application, 
you need to set up a folder under Microsoft IIS 
(Internet Information Server), enable the folder 
for ASP using Internet Services Manager, 
and copy the sample files to that folder.  
Then, view http://localhost/<path>/WmiTest_Main.htm 
using Internet Explorer or other web browser.  

From the main page, WmiTest_Main.htm, you can 
view or modify the configuration objects for 
the MSMQ-MQSeries Bridge.  Functions included 
are:  New Instance, Next Instance, Previous Instance, 
Save Instance, Delete Instance.
