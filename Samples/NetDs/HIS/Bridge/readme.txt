MSMQ-MQSeries Bridge Samples
============================

To build the following MSMQ-MQSeries Bridge samples, you need to set MQSeries Environment variables.
EPRECV
EPSEND
MQSRRecv
MQSRSend

In your Microsoft Platform SDK Build Environment,

Set MQS_LIB to IBM MQSeries lib directory where MQSeries was installed. 
Set MQS_INC to IBM MQSeries include directory where MQSeries was installed. 

For example, set the following environment variables for building these samples:

set MQS_INC=C:\Program Files\MQSeries Client\tools\c\inc
set MQS_LIB=C:\Program Files\MQSeries Client\tools\lib
