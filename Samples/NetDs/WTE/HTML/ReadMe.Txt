======================================================
Notes for HTML samples
======================================================

The HTML samples section containes 2 samples:

1. "Simple" - a simple application demonstrating the basic
   functionality of Web Telephony Engine. Its home page is
   Demo.htm
2. "Restaurant" - A demo of a restaurant page that can be 
   used by both graphical browsers and WTE callers. Also
   shows speech recognition and scripting. Its home page is
   Default.htm

To use one of these samples, point a Web Telephony Application
to its home page, associate a line to the application and call
in.
