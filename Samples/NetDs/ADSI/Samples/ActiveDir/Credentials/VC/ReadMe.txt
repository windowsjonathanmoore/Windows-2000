//
//  Credentials
//
/////////////////////////////////////////////////////////////

This program will allow the user to provide alternate credentials to logon the the current domain.
The username can be: 
 1) NT 4.0 Style domain - e.g  MYDOMAIN\MyUser
 2) Windows 2000 User Principal Name - e.g myUser@mydomain.com
 3) The user name without the domain domain. It will assume the current domain. e.g Administrator
