----------------------------------------------------------------------------
------ Active Directory Sample: User Manager: UsrMgr.EXE --------------------
----------------------------------------------------------------------------

UsrMgr is a sample tool similar to NT 4.0 User Manager with reduced functionalities. With this 
program, you're allowed to browse users, computers,groups for a given domain or a given computer. 
It also allows you to vie the user and group information. 

In the user property, the program displays description, password and account characteristics, groups 
in which the user is member of.

In the group property, the program displays the group's memberships, group's type. 

This sample uses WinNT provider exclusively.
