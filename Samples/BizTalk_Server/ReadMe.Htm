Microsoft BizTalk Server 2000 Samples and Tools

The Microsoft® BizTalk™ Server 2000 sample files, tools, and associated readmes must be used with a running installation of BizTalk Server 2000. If you want to evaluate the samples and tools, you can download a copy of BizTalk Server 2000 Evaluation Edition from the Microsoft BizTalk Server Web site (http://www.microsoft.com/biztalk). The Evaluation Edition contains all the BizTalk Server sample files and tools.
