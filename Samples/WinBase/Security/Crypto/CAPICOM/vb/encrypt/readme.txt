CAPICOM sample Encrypt
-----------------------
Date:  January 15, 2001

The source code in this directory creates a program to encrypt a block of data using a 
semetric key derived from a password. It then demonstrates decrypting the data using the same 
secret password. 

    
This demonstration program runs on Windows 9x with IE 5.0 or later, Windows NT 4.0 with
SP4 or later, Windows 2000 and Whistler.  It uses CAPICOM.dll an ActiveX component, that must 
be registered.


To run this demonstration program, all of the files must be in the same directory and Visual Basic 
must be available. Double click on the project file, encrypt demo.vbp.

This deomstration program uses the following files:
 
readme.txt                 -- This file

auxillary functions.bas    -- code for auxillary functions

encrypt demo.frm           -- Start Form definition

encrypt demo.vbp           -- Visual Basic Project file
