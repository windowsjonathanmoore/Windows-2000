Keyboard Layout DLLs

[This is preliminary documentation and subject to change.]

SUMMARY

These keyboard layout DLLs are compliant with Windows® 2000. The layout DLLs work on both x86 and Alpha platforms, and is 64-bit compliant.

The sample keyboard layout DLLs includes US-English (kbdus), Germen (kbdgr), French (kbdfr), Japanese 101 (kbd101) and Japanese 106 (kbd106). Japanese keyboard layout samples are located under fe_kbds/jpn subdirectory.

BUILDING THE SAMPLE

The files necessary to build the kbdus sample include kbdus.c, kbdus.h, kbdus.rc, kbdus.def, makefile and sources.  Once built, the sample produces a keyboard layout DLL (kbdus.dll). One binary is produced for each subdirectory.

When built from the top of layout directory, kbdus.dll, kbdgr.dll, kbdfr.dll, kbd101.dll and kbd106.dll are built.

This sample is the complete source code for the keyboard layout DLLs work on Windows 2000 systems. To load a customized layout DLLs, rather than the one shipped, you will need to replace it in the %windir%\System32 directory.

The Inf file for various keyboard layout DLLs shipped with Window 2000 system can be found as %windir%\Inf\keyboard.inf.
