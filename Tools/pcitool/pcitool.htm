PCI Configuration Utility

[This is preliminary documentation and subject to change.]

SUMMARY

The PCI Configuration control panel applet allows you to easily view and save the configuration information of all the PCI devices on your system. This utility works on both Microsoft® Windows NT® Version 4.0 and Microsoft Windows® 2000.

RUNNING THE TOOL

To install the tool, first close the Control Panel. It is important to do this before running Setup. Then run Setup.exe.

Known Issues

This tool does not allow you to change every PCI register. The following registers are read-only registers.

  • Vendor ID
  • Device ID
  • Revision ID
  • Sub Class Code
  • Class Code
  • Header Type
  • Subsystem Vendor ID
  • Subsystem ID
  • Reserved
  • Interrupt Pin
  • Min_Gnt
  • Max_Lat

In addition, on x86 systems, this tool cannot update the registers on PCI bridge type devices.

Lastly, this tool recognizes only a limited number of devices. If you see either VenID=xxxx or DevID=xxxx instead of a recognizable vendor name or device name in the list box, this means the tool is not aware of the vendor or the device. If you wish to have the vendor or the device recognized by the tool, please send the following information for each device to nt-ihv@compaq.com.

  • Vendor ID
  • Device ID
  • Subsystem Vendor ID
  • Subsystem ID
  • Vendor Name (30 characters or less)
  • Device Name (30 characters or less)

RESOURCES

PCI Configuration Utility is updated on a quaterly basis, or as often as the vendor ID database on the PCISIG Web site is updated. You can find the latest version of PCI Configuration Utility at http://www.compaq.com/support/files/alphant/drivers/index.html.

To report a problem or make suggestions for this tool, send mail to nt-ihv@compaq.com.
