WHQL Display Monitor EDID Test Tool

[This is preliminary documentation and subject to change.]

SUMMARY

The Display Monitor EDID Test Tool performs EDID verification for all display monitors on the Microsoft® Windows® 2000 platform.

The tool queries the EDID to determine the EDID version, format, values and content, and then returns pass, fail or warning flags per test case. EDID then assesses this information and returns an overall test status of pass or fail.

Once the test is complete, the tool produces a log for each test case, each individual result, and an overall end summary. For each test case, the log states the test case and in the case of warnings or failures and provides a brief explanation of the problem encountered. The tool reads EDID structure versions 1.x and 2.0.

This version of the tool is 32-bit x86 compatible. It has not been tested in a 64-bit environment or on the Alpha platform.

The tool supports the following test cases:

1. EDID Checksum Validation.

2. Product Code Validation (checking for non-zero product code).

3. EDID version check.

4. Verify DPMS Suspend State Support.

5. Verify DPMS Active Off Support.

6. Verify Support for 640x480 @75Hz (required in all monitors).

7. Verify Support for 800x600 @75Hz (required in 14-inch or larger monitors).

8. Verify Support for 1024x768 @75Hz (required in 17-inch or larger monitors).

9. Verify Support for 1280x1024 @75Hz (required in 19-inch or larger monitors).

10. Verify Standard timing.

11. Verify Detailed timing.

BUILDING THE SAMPLE

The DDK includes the EDIDW2K executable. No build is required.
